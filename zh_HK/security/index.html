是什麼讓OneKey贏得了頂級專業機構的信任

OneKey在不影響可訪問性和日常操作的情況下為加密資產提供一流的安全性。未來總是不可預測的，我們的使命是讓每個人都有能力保護他們的加密資產安全。
開源

我們在 Github 上將原始軟件和固件代碼開源，任何人都可以檢查和驗證它。

我們相信，只有這樣，我們才能澄清自己，以確保OneKey的道德受到群眾的審查。

安全元件

我們觀察到，有些錢包將最有價值的私鑰直接存儲在微處理器（MCU）中，這是相當可怕的。

相比之下， OneKey使用安全元件來保護您的加密資產——確切地說，是來自 Microchip 的 ATECC608A。

通過精心設計的SHA-256響應控制，我們可以有效防止重放和竊聽攻擊。

冷鍵

我們不會將私鑰存儲在您的設備微處理器 (MCU) 中，而是存儲在單獨的安全芯片 ATECC608A 中。

同樣，無論您如何使用OneKey硬件錢包，您的私鑰自始至終都沒有接觸過互聯網。

具體來說， OneKey幫助您確認和授權每筆鏈上交易，簽名過程是在您的設備上完成的，而不是您的 PC 或手機。這可以防止惡意程序欺騙或欺騙您進行高風險操作：例如將加密資產發送到壞人的偽裝地址。
